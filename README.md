
## Basic template for developing Co-pilot extension. 

This template provides Co-pilot light dimensions with mounting holes and GPIO pin header for controlling other peripherals.

How to use this template with KiCad 7.0 ?

To use this template just clone this repo in user template folder `~/.local/share/kicad/7.0/template`

To user this template you need to:

`File -> New Project From Template`

There should be two tabs "System Templates" and "User Templates".
You want to select `User Templates`.
Select copilot template and Enjoy!
