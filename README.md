
# Copilot Lite - 3xUSB 2.0 extension board

![Extension board](images/board_top.jpg)

This boards mounts on top of the Copilot Lite board, like a Raspberry Pi hat or an Arduino shield.  
It provides 3 additional USB 2.0 passthrough channels which can be controlled by a Copilot Lite GPIO.  
The GPIO controlling each channel can be configured through jumpers.
